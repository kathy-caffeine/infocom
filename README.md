# Тестовое задание

## Формат используемых данных:

### Номер машины: 
`XdddXX`, где `X` - заглавная латинская буква, `d` - цифра
### Веса
Число, до 3 знаков после запятой. В качестве разделителя целой и дробной части может использоваться как `.`, так и `,`, или ничего:
* `1.0`
* `1,0`
* `1`
### Даты
`yyyy-MM-dd`
